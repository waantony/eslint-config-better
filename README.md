# eslint-config-better

[toc]

Better ESLint configs, JavaScript Better Style, ESLint Shareable Config

## install

```bash
npm i -D eslint-config-better
```

## extends in `.eslintrc.js`

```js
module.exports = {
  extends: [
    'better', // or eslint-config-better
  ],
  rules: {
    // ...other custom rules
  }
}

```
